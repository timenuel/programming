Bagian-bagian Komputer

Komputer terbagi menjadi beberapa bagian, diantaranya :

Hardware (Perangkat Keras)

Hardware atau perangkat keras adalah bagian Komputer yang bisa kita lihat wujud fisiknya

Contoh Hardware :

Monitor, Keyboard, Mouse

Software (Perangkat Lunak)

Perangkat Lunak atau software adalah kumpulan perintah yang dijalankan komputer untuk menyelesaikan pekerjaan.

Contoh Software :

Browser, Aplikasi Pengolah Kata, Pemutar Lagu/Video
